# Boss Hiring App

Business social network App.

> Frontend -> React, Redux and Ant Design Mobile UI

> Backend -> Express.js, socket.IO and MongoDB

## Features
- Recruiters/Job seekers signup and signin
- Chat between recruiters and job seekers
